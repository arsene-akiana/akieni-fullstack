# akieni-fullstack

A collection of practical assignments and projects completed during my Full Stack Developer training at **Akieni Academy** (Brazzaville, Congo).

This repository is updated continuously as I progress through the program. Each folder corresponds to a technology or module covered during training.

---

## 📁 Repository Structure

```
akieni-fullstack/
│
├── html/           # Semantic HTML5 exercises and pages
├── css/            # Styling, layouts, responsive design
├── javascript/     # DOM manipulation, logic, algorithms
├── react/          # Component-based UI with React
├── nodejs/         # Server-side development with Node.js
└── ...             # More technologies to come
```

---

## 🛠️ Technologies Covered

- **HTML5** — Semantic structure and accessibility
- **CSS3** — Flexbox, Grid, animations, responsive design
- **JavaScript** — ES6+, DOM, events, asynchronous programming
- **React** — Components, hooks, state management
- **Node.js** — REST APIs, server logic, npm ecosystem
- *And more to come as the training progresses...*

---

## 🎯 Objectives

- Build a strong foundation in modern full stack web development
- Deliver clean, well-structured, and readable code
- Document learning progress through real, hands-on projects
